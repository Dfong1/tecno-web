Water QA

Water QA es un proyecto desarrollado con multiples tecnologías, con la capacidad de detectar la calidad correcta del agua.


El principal objetivo del proyecto es proporcionarle a los usuarios una interfaz clara, entendible y bonita para la información proporcionada por los sensores


Angular

TypeScript

Laravel

Swift

    Lenguajes:

  • TYPESCRIPT
  • PHP
  • MONGO-DB
  • SWIFT
  • ARDUINO

    Tecnologias:

  • LARAVEL
  • ANGULAR
  • ARDUINO
  • XCODE

Los lenguajes y tecnologias implementadas fueron la clave para poder hacer una aplicación web y una aplicación para dispositivos iOS


El proposito de implementar el proyecto en estos dispositivos es para que cualquier usuario pueda ver los registros de los sensores, para que pueda saber la calidad del agua y así darle el tratamiento adecuado.
